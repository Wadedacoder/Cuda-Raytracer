# Cuda-Raytracer
A raytracer made in cuda using Raytracing in One Weekend as a reference.

## Features
- [x] Sphere intersection
- [x] Linear interpolation
- [x] Ray-sphere intersection
- [x] Speedup using CUDA
- [x] Stored as pngs
- [] Diffuse materials
- [] Metal materials
- [] Anti-aliasing

## Usage
```bash
    make
    ./a.exe
```
